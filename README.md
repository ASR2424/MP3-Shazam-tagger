# MP3 AUTO Tagger

Windows-приложение для автоматического тегирования MP3 файлов с помощью распознавания музыки через Shazam.

Программа умеет:
- распознавать песни через Shazam
- автоматически записывать ID3 теги
- скачивать обложки высокого качества
- загружать lyrics из LRCLIB
- редактировать теги вручную
- пакетно обрабатывать MP3 файлы
- переименовывать файлы по шаблону

---

# Возможности

## 🎵 Автоматическое распознавание песен

Определяет:
- Название трека
- Исполнителя
- Альбом
- Жанр
- Год

используя сервис Shazam.

---

## 🏷 Редактирование ID3 тегов

Записывает:
- `TIT2` (Title)
- `TPE1` (Artist)
- `TALB` (Album)
- `TCON` (Genre)
- `TDRC` (Year)
- `USLT` (Lyrics)
- `APIC` (Cover Art)

---

## 🖼 Работа с обложками

- скачивание обложек из Shazam
- поддержка оригинального качества
- автоматическое изменение размера
- сравнение существующих и найденных обложек
- предпросмотр внутри интерфейса

---

## 📝 Поддержка Lyrics

Загружает текст песен из LRCLIB:
- synced lyrics
- plain text lyrics

---

## ⚙ Умные режимы замены тегов

Поддерживаются режимы:
- заменять всегда
- заменять только пустые теги
- спрашивать перед заменой

Такая же логика доступна и для замены обложек.

---

## 📂 Пакетная обработка

- выбор нескольких файлов
- обработка папок
- progress bar
- кнопка остановки
- асинхронная обработка

---

## ✏ Ручной редактор тегов

Встроенный редактор позволяет:
- вручную изменять теги
- редактировать lyrics
- просматривать встроенную обложку
- просматривать найденную обложку

---
